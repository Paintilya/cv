Bloometti

Un robot sur l'application de messagerie Discord développé en JavaScript par le biais de Node.JS. Ce robot permet d'ajouter un système de niveaux similaire à celui d'un jeu vidéo dans les serveurs (groupes) Discord où il est installé. Les points d'expérience sont gagnés en envoyant des messages dans ces serveurs. Ensuite, il est possible pour l'utilisateur d'envoyer une commande sur Discord pour interagir avec le robot. (Voir image)

Librairies utilisées

  • Discord.JS, permettant de créer un robot Discord ainsi que de le faire interagir avec l'API de Discord.
  • PM2, permettant de gérer l'exécution de l'application.
  • Canvas, permettant de manipuler des images.
  • MongoDB, pour interagir avec la base de données MongoDB

Technologies utilisées

  • JavaScript
  • Node.JS
  • JSON
  • MongoDB
  • Serveur Ubuntu
  • Site Web (HTML, CSS)

Concepts utilisés

  • Programmation modulaire
  • Développement web frontend et responsive
  • Administration de base sous Linux
  • Bases de données
